# 🌾 AI Crop Yield Prediction Dashboard

This project predicts crop yield using machine learning based on rainfall, fertilizer usage, pesticide usage, and regional parameters.

## 🚀 Live Demo
(https://ai-crop-yield-prediction.streamlit.app/)

## 📊 Features
- Multi-crop yield prediction
- State & season-based analysis
- Agricultural insights dashboard
- Scenario simulation (what-if analysis)
- Data visualization (rainfall vs yield, state-wise trends)

## 🧠 Machine Learning
- Model: Random Forest Regressor
- Features:
  - Rainfall
  - Fertilizer
  - Pesticide
  - State
  - Season

## 📂 Dataset
- Source: Kaggle
- Contains multi-year agricultural data across Indian states

## ⚙️ How to Run

1. Install dependencies
pip install -r requirements.txt

2. Train model
python train_model.py

3. Run app
streamlit run app.py
